# Sistem Pengadaan dan Permintaan Barang

Sistem Pengadaan dan Permintaan Barang adalah aplikasi berbasis web yang dirancang untuk mempermudah proses pengajuan permintaan barang dan pengelolaan pengadaan barang dalam suatu organisasi atau instansi. Sistem ini membantu mencatat, menyetujui, dan memproses permintaan hingga pengadaan secara terstruktur dan efisien.
